## JJwt를 이용한 자바에서 JWT사용하기
다음과 같이 사용이 가능하다.

### 토큰 생성
```java
String jwt = Jwts.builder()
				.setSubject("")
				.claim("name", "Chang")
				.claim("id", 1)
				.signWith(
						SignatureAlgorithm.HS256,
						"secret".getBytes()
				)
				.compact();
```
### 토큰 파싱
```java
    Jws<Claims> claims = Jwts.parser()
            .setSigningKey(secret.getBytes("UTF-8"))
                            .parseClaimsJws(token);
    
    String item = claims.getBody()
                    .get(target)
                    .toString();
```
